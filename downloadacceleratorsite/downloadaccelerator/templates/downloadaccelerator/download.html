Installation Instructions

You must have a linux distribution installed on your computer and the browser FireFox installed.
After that, install Download Accelerator module integration of FireFox.
Now you have two possibilities:

Installation by debian package

  • Download the gpg public key which you use to verify the authenticity of debian package here
  • Import the key by typing gpg --import downloadaccelerator-VERSION.gpg on terminal
  • Get a debian package corresponding to the gpg public key version down below
    • downloadaccelerator-3.1
  • Verify that the debian package is the authentic one by typing dpkg-sig --verify downloadaccelerator-VERSION.deb to the terminal
  • If the package is authentic, install it by typing dpkg -i downloadaccelerator-VERSION.deb on terminal
  • Restart computer
  • Download Accelerator is ready

Or

Installation by snapstore

  • If you do not have snapstore installed, install it by typing on your favourite terminal sudo snap install snap-store on Ubuntu or snap install snap-store in superuser on others distributions.
  • Install the Download Accelerator snap from snap-store.
  • Launch Download Accelerator. This step is very important, to let the configuration of Download Accelerator snap.
  • Restart your user session
  • Once you are restart your session, Download Accelerator is ready to be used.
